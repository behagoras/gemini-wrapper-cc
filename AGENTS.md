# gemini-plugin-cc

Claude Code plugin that delegates work to Google's Gemini CLI (reviews, second opinions, web-grounded research, task delegation) — the Gemini analog of `codex-plugin-cc`. Structure: `agents/`, `commands/`, `skills/`, `scripts/`. Full docs: [`README.md`](README.md); shared workspace rules: root [`AGENTS.md`](../AGENTS.md).
